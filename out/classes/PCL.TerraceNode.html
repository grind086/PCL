PCL.TerraceNode Class

Extends PCL.ModifierNode
Defined in: src/extras/modifiers/TerraceNode.js:1
Module: PCL

Maps an input to a curve defined by the given control points. Creates a terracing effect. See http://libnoise.sourceforge.net/docs/classnoise_1_1module_1_1Terrace.html#_details for examples (todo: make our own examples).

Based on Jason Bevin's Terrace module from libnoise.
